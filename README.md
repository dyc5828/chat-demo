## Get Started

1. Install `nodemon` globally

```shell
$ [sudo] npm install --global nodemon
``` 


2. Install local npm packages

```shell
$ cd <path/to/project>
$ npm install
```

3. Run express server

```shell
$ nodemon app.js
```
